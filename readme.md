# Java 21, Spring boot 3.5.7 연습 프로젝트

## 프로젝트의 목적
Java 및 Spring boot 에서 기본 및 심화 기술의 응용력을 높히기 위함

이로 인해 실무에서 자연스러운 코드 활용 및 역량 강화를 목표함

## 프로젝트의 방향성
- 보안 기술 적용
  - 프로젝트 주제에 걸맞는 보안 기술 적용 및 공부
- 대규모 트래픽 상정
  - 서버 부하 분산 기술 및 대처법 공부
  - 제한된 서버 리소스를 활용하는법 공부
- 무결성 지향
  - 단순한 기술 구현에서도 다양한 상황을 고려해 버그가 일어나지 않도록 구현
- AWS 및 개인 서버 활용력 강화
  - 실무를 대비한 AWS 서비스 활용법 공부
  - 개인 서버용 리눅스 PC를 통한 프로젝트 구축
